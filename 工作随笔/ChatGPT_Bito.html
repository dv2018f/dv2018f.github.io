概要

Bito是一款在IntelliJ IDEA编辑器中的插件，Bito插件是由ChatGPT团队开发的，它是ChatGPT团队为了提高开发效率而开发的一款工具。

# 一、Bito介绍

# 1.1 什么是Bito？

​ Bito是一款在IntelliJ IDEA编辑器中的插件，Bito插件是由ChatGPT团队开发的，它是ChatGPT团队为了提高开发效率而开发的一款工具。Bito插件的强大之处在于它可以帮助开发人员更快地提交代码，同时还提供了一些有用的功能，如自动补全提交信息、快速查看历史记录等。Bito 的 AI 帮助开发人员显著加快其影响力。这是一把瑞士军刀的功能，使用与 ChatGPT 相同的模型，可以提高您的开发人员生产力并每天为您节省一个小时！

# 1.2 为什么要使用Bito？

​ Bito AI使编写代码，理解语法，编写测试用例，解释代码，评论代码，检查安全性甚至解释高级概念变得容易。经过数十亿行代码和数百万个文档的培训，我们可以帮助您完成的工作，而无需搜索网络或浪费时间在繁琐的事情上，这真是令人难以置信。

# 1.3 Bito 模型

​ Bito使用哪些模型来生成响应？Bito使用GPT-4和GPT Turbo 3.5型号的组合。长的、复杂的提示被路由到GPT-4，短的提示被发送到Turbo 3.5型号。这有助于Bito平衡其产生的输出质量和为用户服务的成本。

# 1.解释代码

 public ResultResponse<ApproveApplyResDTO> apply(@RequestBody ApproveApplyRequest request)
            throws CrmException {
        ApproveApplyResDTO response;

        AuthUserInfo user = getSessionUser();
        String param;
        int enterApply = 0;
        if (request.getType() > 0) {
            enterApply = 1;
        }
        Long saleId;
        int type;
        if (request.getChangeStatusParams() != null) {
            // 订单状态
            saleId = request.getChangeStatusParams().getSaleId();
            request.getChangeStatusParams().setStatusId(
                    ObjectUtils.defaultIfNull(request.getChangeStatusParams().getStatusId(),
                            SaleMoneyConstants.AGREE_STATUS
                    ));
            param = JSONObject.toJSONString(request.getChangeStatusParams());
            type = ApproveTypeEnum.SALE_APPROVE.getCode();
        } else if (request.getBatchPaymentRecordParams() != null &&
                CollectionUtils.isNotEmpty(request.getBatchPaymentRecordParams())) {
            // 批量回款
            saleId = request.getBatchPaymentRecordParams().get(0).getSaleId();
            if (saleId == null) {
                SalePaymentCollectionRecordOperateVO vo = request.getBatchPaymentRecordParams().get(0);
                if (vo.getRecordId() == null || vo.getRecordId() == 0) {
                    SalePaymentDetailVO plan = salePaymentPlanService
                            .getDetailByPlanId(user.getCorpId(), user.getUserId(),
                                    request.getBatchPaymentRecordParams().get(0).getPlanId());
                    if (plan != null) {
                        saleId = plan.getSaleId();
                    }
                } else {
                    SalePaymentCollectionRecordOperateVO hisRecord = salePaymentRecordQueryService.getById(getCorpId(), vo.getRecordId());
                    if (hisRecord != null) {
                        saleId = hisRecord.getSaleId();
                    }
                }
            }
            if (saleId == null) {
                throw ExceptionEnums.PARAM_INVALID.getException();
            }
            PaymentRecordBatchOperateVO recordBatchOperateVO = new PaymentRecordBatchOperateVO();
            recordBatchOperateVO.setSaleId(saleId);
            recordBatchOperateVO.setRecords(new ArrayList<>());
            for (SalePaymentCollectionRecordOperateVO recordParam : request
                    .getBatchPaymentRecordParams()) {
                PaymentRecordOperateVO recordOperateVO = new PaymentRecordOperateVO();
                BeanUtils.copyProperties(recordParam, recordOperateVO);
                recordBatchOperateVO.getRecords().add(recordOperateVO);
            }
            param = JSONObject.toJSONString(recordBatchOperateVO);
            type = ApproveTypeEnum.PAYMENT_APPROVE.getCode();
        } else {
            throw ExceptionEnums.PARAM_INVALID.getException();
        }

        List<ApproveProcessRequest> processRequests = request.getProcess();
        List<Long> copyUsers = request.getCopyUsers();
        ApproveProcessVO process = new ApproveProcessVO();
        // 审批人
        if (CollectionUtils.isNotEmpty(processRequests)) {
            process.setProcessUsers(processRequests.stream().map(processRequest -> {
                ApproveProcessUserVO userVO = new ApproveProcessUserVO();
                userVO.setNode(processRequest.getNode());
                userVO.setApproveUsers(processRequest.getApproveUsers().stream()
                        .map(e -> {
                            SaleApproveUserInfoDTO userInfo = new SaleApproveUserInfoDTO();
                            userInfo.setId(e);
                            return userInfo;
                        }).collect(Collectors.toList()));
                return userVO;
            }).collect(Collectors.toList()));
        }
        // 抄送人
        if (CollectionUtils.isNotEmpty(copyUsers)) {
            process.setCopyUsers(copyUsers.stream().map(e -> {
                SaleApproveUserInfoDTO copyUser = new SaleApproveUserInfoDTO();
                copyUser.setId(e);
                return copyUser;
            }).collect(Collectors.toList()));
        }
        response = approveService
                .apply(user.getCorpId(), user.getUserId(), enterApply, type, saleId, param, process,
                        request.getContent(), request.getApplyId());
        response.setSaleId(saleId);
        return new ResultResponse<>(response).getResult(CommResponse.SUCCESS_DO);
    }

# 2.代码的解释翻译
这段代码用于申请审批。它首先从会话中获取用户信息，然后检查请求类型是否大于0。如果是，它将enterApply设置为1。然后它检查changeStatusParams是否为null，如果是，则将saleId设置为changeStatusParams的saleId，并将statusId设置为AGREE_STATUS。然后使用changeStatusParams创建一个参数字符串，并将类型设置为SaleApprove。如果batchPaymentRecordParams不为null且不为空，它将saleId设置为第一个元素的saleId，并创建一个PaymentRecordBatchOperateVO对象，其中包含saleId和记录。然后使用PaymentRecordBatchOperateVO创建一个参数字符串，并将类型设置为PaymentApprove。然后创建一个ApproveProcessVO对象，其中包含processUsers和copyUsers，并调用approveService的apply方法，其中包括用户的corpId、userId、enterApply、type、saleId、param、process、content和applyId。然后将响应的saleId设置为saleId，并返回一个带有响应和成功状态的ResultResponse。  
逐步解释： 
1. 声明一个类型为ApproveApplyResDTO的响应变量 
2. 从会话中获取用户信息 
3. 声明一个参数字符串、enterApply整数、saleId长整数和类型整数 
4. 检查请求类型是否大于0，如果是，则将enterApply设置为1 
5. 检查changeStatusParams是否为null，如果是，则将saleId设置为changeStatusParams的saleId，并将statusId设置为AGREE_STATUS。使用changeStatusParams创建参数字符串，并将类型设置为SaleApprove 
6. 检查batchPaymentRecordParams是否不为null且不为空，如果是，则将saleId设置为第一个元素的saleId，并创建一个包含saleId和记录的PaymentRecordBatchOperateVO对象。使用PaymentRecordBatchOperateVO创建参数字符串，并将类型设置为PaymentApprove 
7. 创建一个包含processUsers和copyUsers的ApproveProcessVO对象 
8. 调用approveService的apply方法，包括用户的corpId、userId、enterApply、type、saleId、param、process、content和applyId 
9. 将响应的saleId设置为saleId 
10. 返回一个带有响应和成功状态的ResultResponse

# **3.代码的优化提问 **

从入参的角度去简化这个方法，比如可以直接传saleId, enterApply，param可以通过type获取，process 也可以从前端传参获取，审批人和抄送人可以直接前端传参获取，无需转换。根据这些入参重新定义ApproveApplyRequest 参数并简化该代码，提高可读性

简化后的方法如下： 
 Human:public ResultResponse<ApproveApplyResDTO> apply(Long saleId, int enterApply, int type, String param,  
        List<ApproveProcessRequest> processRequests, List<Long> copyUsers, String content, Long applyId)      
        throws CrmException {      
    ApproveApplyResDTO response;      
 
    AuthUserInfo user = getSessionUser();      
      
    List<ApproveProcessVO> process = new ArrayList<>();      
    // 审批人      
    if (CollectionUtils.isNotEmpty(processRequests)) {      
        process = processRequests.stream().map(processRequest -> {      
            ApproveProcessVO approveProcessVO = new ApproveProcessVO(); 
            ApproveProcessUserVO userVO = new ApproveProcessUserVO();      
            userVO.setNode(processRequest.getNode());      
            userVO.setApproveUsers(processRequest.getApproveUsers().stream()      
                    .map(e -> {      
                        SaleApproveUserInfoDTO userInfo = new SaleApproveUserInfoDTO();      
                        userInfo.setId(e);      
                        return userInfo;      
                    }).collect(Collectors.toList()));      
            approveProcessVO.setProcessUsers(Collections.singletonList(userVO)); 
            return approveProcessVO; 
        }).collect(Collectors.toList());      
    } 
    // 抄送人      
    List<SaleApproveUserInfoDTO> copyUserList = null; 
    if (CollectionUtils.isNotEmpty(copyUsers)) {

            SaleApproveUserInfoDTO copyUser = new SaleApproveUserInfoDTO();      
            copyUser.setId(e);      
            return copyUser;      
        }).collect(Collectors.toList());      
    }      
    response = approveService      
            .apply(user.getCorpId(), user.getUserId(), enterApply, type, saleId, param, process,      
                    content, applyId, copyUserList);      
    response.setSaleId(saleId);      
    return new ResultResponse<>(response).getResult(CommResponse.SUCCESS_DO);      
}

# 场景二： 优化重构代码

public ApproveApplyResDTO apply(long corpId, long userId, int enterApply, int type, Long saleId,
            String param, ApproveProcessVO approveProcessVO, String content, Long applyId)
            throws CrmException {
        ApproveApplyResDTO res = new ApproveApplyResDTO();
        res.setStatus(ApproveConstants.CHECK_PARAM_ERROR_APPROVE.getStatus());
        res.setCode(ApproveConstants.CHECK_PARAM_ERROR_APPROVE.getCode());
        res.setMessage(ApproveConstants.CHECK_PARAM_ERROR_APPROVE.getMessage());

        // 2.检验是否需要审批
        ApproveCheckDTO check = applyCheck(corpId, userId, enterApply, type, content, applyId,
                param,saleId);
        log.info("check ={}，param = {}", check,param);

        // 3.审批流程校验
        // 不需要审批
        if (check.getStatus().equals(ApproveConstants.APPROVE_NO_NEED)) {
            res.setStatus(ApproveConstants.APPROVE_NO_NEED);
            res.setCode(ApproveConstants.SUCCESS_DO);
            res.setMessage(check.getMessage());
            res.setData(check.getData());
            res.setType(check.getType());
            res.setSubType(check.getSubType());
            return res;
        }
        // 需要审批
        if (enterApply == Constants.NO && check.getStatus().equals(ApproveConstants.APPROVE_NEED)) {
            res.setCode(ApproveConstants.SUCCESS_DO);
            res.setStatus(ApproveConstants.APPROVE_NEED);
            res.setType(check.getType());
            res.setSubType(check.getSubType());
            res.setMessage(Constants.EMPTY_STRING);
            return res;
        }
        // 参数错误，审批中
        if (check.getStatus().equals(ApproveConstants.APPROVE_PARAM_ERROR) ||
                check.getStatus().equals(ApproveConstants.BUSINESS_APPROVE_ING) ||
                check.getStatus().equals(ApproveConstants.BUSINESS_APPROVE_ERROR)) {
            if (check.getApply() != null) {
                res.setApplyId(check.getApply().getApplyId());
            }
            res.setType(check.getType());
            res.setSubType(check.getSubType());
            res.setStatus(check.getStatus());
            res.setCode(check.getCode());
            res.setMessage(check.getMessage());
            return res;
        }

        // 进入审批申请主流程
        if (enterApply == Constants.YES && check.getStatus()
                .equals(ApproveConstants.APPROVE_NEED)) {

            // 4.获取审核规则
            ApproveRuleDTO approveRule = saleApproveRuleService
                    .getRuleByCorpIdAndTypeId(corpId, type, check.getSubType());

            log.info("approveRule = {}",approveRule);

            if(approveRule == null){
                return res;
            }

            // 5.计算审批人和抄送人
            approveProcessVO = applyComponent.computeApproveUser(corpId, userId, approveRule,
                    approveProcessVO);

            log.info("审批人信息 approveProcessVO：{} ", approveProcessVO);
            // 6.写入审批申请记录
            SaleApproveApply apply;
            apply = applyComponent.genSaleApproveApply(corpId, userId, saleId,
                    approveRule, approveProcessVO, content);
            log.info("审批申请信息 apply：{} ", apply);
            List<SaleApproveRecord> records;
            if (apply == null) {
                res.setType(check.getType());
                res.setSubType(check.getSubType());
                return res;
            }
            // 7.获取审批记录
            if (applyId == null || applyId == 0) {
                records = applyComponent
                        .genApproveRecord(corpId, apply, approveProcessVO);
            } else {
                records = applyComponent
                        .genReApplyRecord(corpId, apply, approveRule, approveProcessVO, applyId);
            }
            log.info("生成了审批记录records：{} ", records);

            // 无审核人（上级模式），全部离职了，或审批人全部自动同意
            ApproveHandleResDTO handle = null;
            if (CollectionUtils.size(records)
                    == records.stream().map(SaleApproveRecord::getUserId)
                    .filter(id -> id == 0).count()) {
                ApproveTypeEnum approveTypeEnum = ApproveTypeEnum.codeOf(type);
                AbstractApproveService businessApprove = applicationContext
                        .getBean(approveTypeEnum.getBeanName(), AbstractApproveService.class);
                handle = businessApprove.handle(corpId, userId, saleId, param, null);
                apply.setApplyStatus(SaleApproveApplyEnum.APPROVE_AGREE.getCode());
                if (handle != null) {
                    res.setData(handle.getData());
                }
                res.setStatus(ApproveConstants.APPROVE_NO_NEED);
                res.setCode(ApproveConstants.SUCCESS_DO);
                res.setMessage(check.getMessage());
                res.setData(check.getData());
                res.setType(check.getType());
                res.setSubType(check.getSubType());
                return res;
            }

            saleApproveApplyDao.insert(corpId, apply);
            applyId = apply.getApplyId();
            saleApproveRecordDao.batchInsert(corpId, records);
            // 8.计算展示数据
            ApproveTypeEnum approveTypeEnum = ApproveTypeEnum.codeOf(type);
            AbstractApproveService businessApprove = applicationContext
                    .getBean(approveTypeEnum.getBeanName(), AbstractApproveService.class);

            List<ApplyDataDTO> applyDatas = businessApprove
                    .computeApplyData(corpId, userId, param);

            log.info("展示数据1 param = {} applyDatas = {}", param, applyDatas);
            // 9.审批申请预处理：数据入库

            applyDatas = businessApprove.preHandle(corpId, userId, param, applyDatas, saleId);
            // 10.写入扩展表记录
            log.info("展示数据2 applyDatas = {}", applyDatas);

            saleMoneyApproveExtendDao.batchInsert(corpId, genApplyExtend(
                    corpId, param, apply, applyDatas));
            // 11，审批消息
            List<SaleApproveRecord> recordMsg = records.stream()
                    .filter(c -> c.getArrive().equals(
                            SaleApproveArriveEnum.IS_ARRIVE.getCode()))
                    .collect(Collectors.toList());
            for (SaleApproveRecord record : recordMsg) {
                sendMessageComponent
                        .sendNewApprovalImNsq(corpId, apply.getUserId(), record.getUserId(),
                                apply,
                                record.getId(),
                                SaleApprovalIMEnum.APPROVE_WAITING.getCode());
            }
            // 12.更新业务审批状态
            List<Long> businessIdsUpdate = new ArrayList<>();
            List<Long> businessIdsAdd    = new ArrayList<>();
            for (ApplyDataDTO applyData : applyDatas) {
                if (StringUtils.isBlank(applyData.getOldData())) {
                    businessIdsAdd.add(applyData.getBusinessId());
                } else {
                    businessIdsUpdate.add(applyData.getBusinessId());
                }
            }
            if (CollectionUtils.isNotEmpty(businessIdsUpdate)) {
                businessApprove.updateApproveStatus(corpId, userId, ApproveConstants
                        .BUSINESS_APPROVE_STATUS_UPDATE, businessIdsUpdate);
            }
            if (CollectionUtils.isNotEmpty(businessIdsAdd)) {
                businessApprove.updateApproveStatus(corpId, userId, ApproveConstants
                        .BUSINESS_APPROVE_STATUS_ADD, businessIdsAdd);
            }
            // 13.发送轨迹
            businessApprove.sendApplyContact(corpId, userId, applyDatas);
        }
        res.setStatus(ApproveConstants.APPROVE_SUCCESS);
        res.setCode(ApproveConstants.SUCCESS_DO);
        res.setMessage(Constants.EMPTY_STRING);
        res.setType(type);
        res.setSubType(ObjectUtils.defaultIfNull(check.getSubType(), 0));
        res.setApplyId(applyId);
        return res;
    }

# 1.代码优化提问

​ 简化这个方法，提高可读性，并提出优化的点； 优化方向：这个方法对审核的校验和审核的业务耦合很严重，审核校验可以独立一个方法，审核业务处理单独一个方法。

private ApproveApplyResDTO handleApproveProcess(Long corpId, Long userId, Integer enterApply,
                                                Integer type, String content, Long applyId, Map<String, Object> param,
                                                Long saleId) {
    ApproveApplyResDTO res = new ApproveApplyResDTO();

    // 2.检验是否需要审批
    ApproveCheckDTO check = applyCheck(corpId, userId, enterApply, type, content, applyId,
            param, saleId);
    log.info("check ={}，param = {}", check, param);

    // 3.处理审批结果
    res = handleApproveResult(enterApply, check, corpId, userId, type, content, applyId, param, saleId);
    return res;
}

private ApproveApplyResDTO handleApproveResult(Integer enterApply, ApproveCheckDTO check, Long corpId,
                                               Long userId, Integer type, String content, Long applyId,
                                               Map<String, Object> param, Long saleId) {
    ApproveApplyResDTO res = new ApproveApplyResDTO();

    // 不需要审批
    if (check.getStatus().equals(ApproveConstants.APPROVE_NO_NEED)) {
        res = setApproveResNoNeed(check);
    }
    // 需要审批
    else if (enterApply == Constants.NO && check.getStatus().equals(ApproveConstants.APPROVE_NEED)) {
        res = setApproveResNeed(check);
    }
    // 参数错误，审批中
    else if (check.getStatus().equals(ApproveConstants.APPROVE_PARAM_ERROR) ||
            check.getStatus().equals(ApproveConstants.BUSINESS_APPROVE_ING) ||
            check.getStatus().equals(ApproveConstants.BUSINESS_APPROVE_ERROR)) {
        res = setApproveResErrorOrIng(check);
    }
    // 进入审批申请主流程
    else if (enterApply == Constants.YES && check.getStatus()
            .equals(ApproveConstants.APPROVE_NEED)) {
        res = handleApproveMainProcess(corpId, userId, type, content, applyId, param, saleId, check);
    }

    return res;
}

# 2.2 业务处理方法（提交审批后处理）
public ApproveApplyResDTO commitApply(long corpId, long userId, TriggerApproveDTO triggerApproveDTO)
            throws CrmException {
        // 1.审批参数校验
        ApproveApplyResDTO approveApplyResDTO = trigger(corpId, userId, triggerApproveDTO);

        // 2.参数异常或者不需要审批
        if (!Objects.equals(approveApplyResDTO.getStatus(), ApproveConstants.APPROVE_NEED)) {
            return approveApplyResDTO;
        }

        // 3.业务处理
        // 获取审核规则
        ApproveRuleDTO approveRule = saleApproveRuleService
                .getRuleByCorpIdAndTypeId(corpId, approveApplyResDTO.getType(), approveApplyResDTO.getSubType());

        log.info("commitApply approveRule = {}", approveRule);

        ApproveApplyResDTO resultDTO = new ApproveApplyResDTO();
        resultDTO.setStatus(ApproveConstants.CHECK_PARAM_ERROR_APPROVE.getStatus());
        resultDTO.setCode(ApproveConstants.CHECK_PARAM_ERROR_APPROVE.getCode());
        resultDTO.setMessage(ApproveConstants.CHECK_PARAM_ERROR_APPROVE.getMessage());

        if (approveRule == null) {
            log.warn("can not find rule:{},{}", approveApplyResDTO.getType(), approveApplyResDTO.getSubType());
            return resultDTO;
        }

        // 5.计算审批人和抄送人
        ApproveProcessVO approveProcessVO = applyComponent.computeApproveUser(corpId, userId,
                approveRule, triggerApproveDTO.getProcess());

        log.info("commitApply 审批人信息 approveProcessVO：{} ", approveProcessVO);

        // 6.写入审批申请记录
        SaleApproveApply apply;
        apply = applyComponent.genSaleApproveApply(corpId, userId, triggerApproveDTO.getSaleId(),
                approveRule, approveProcessVO, triggerApproveDTO.getContent());
        log.info("commitApply 审批申请信息 apply：{} ", apply);

        if (apply == null) {
            resultDTO.setType(approveApplyResDTO.getType());
            resultDTO.setSubType(approveApplyResDTO.getSubType());
            return resultDTO;
        }

        // 7.获取审批记录
        List<SaleApproveRecord> records;
        if (triggerApproveDTO.getApplyId() == null || triggerApproveDTO.getApplyId() == Constants.ZERO) {
            records = applyComponent
                    .genApproveRecord(corpId, apply,  approveProcessVO);
        } else {
            records = applyComponent
                    .genReApplyRecord(corpId, apply, approveRule, approveProcessVO, triggerApproveDTO.getApplyId());
        }
        log.info("commitApply 生成了审批记录records：{} ", records);

        saleApproveApplyDao.insert(corpId, apply);
        Long applyId = apply.getApplyId();
        saleApproveRecordDao.batchInsert(corpId, records);


        // 8.计算展示数据
        ApproveTypeEnum approveTypeEnum = ApproveTypeEnum.codeOf(triggerApproveDTO.getType());
        AbstractApproveService businessApprove = applicationContext
                .getBean(approveTypeEnum.getBeanName(), AbstractApproveService.class);

        String param = getParam(approveTypeEnum, triggerApproveDTO);

        List<ApplyDataDTO> applyDataList = businessApprove
                .computeApplyData(corpId, userId, param);

        log.info("commitApply 展示数据1 param = {} applyDataLists = {}", param, applyDataList);


        // 9.审批申请预处理：数据入库
        applyDataList = businessApprove.preHandle(corpId, userId, param, applyDataList, triggerApproveDTO.getSaleId());
        log.info("commitApply 展示数据2 applyDataList = {}", applyDataList);

        // 10.写入扩展表记录
        saleMoneyApproveExtendDao.batchInsert(corpId, genApplyExtend(
                corpId, param, apply, applyDataList));

        // 11，审批消息,找到状态是已到达的记录
        List<SaleApproveRecord> recordMsg = records.stream()
                .filter(c -> c.getArrive().equals(
                        SaleApproveArriveEnum.IS_ARRIVE.getCode()))
                .collect(Collectors.toList());
        for (SaleApproveRecord record : recordMsg) {
            sendMessageComponent
                    .sendNewApprovalImNsq(corpId, apply.getUserId(), record.getUserId(),
                            apply,
                            record.getId(),
                            SaleApprovalIMEnum.APPROVE_WAITING.getCode());
        }

        // 12.发送轨迹
        businessApprove.sendApplyContact(corpId, userId, applyDataList);


        resultDTO.setStatus(ApproveConstants.APPROVE_SUCCESS);
        resultDTO.setCode(ApproveConstants.SUCCESS_DO);
        resultDTO.setMessage(Constants.EMPTY_STRING);
        resultDTO.setType(triggerApproveDTO.getType());
        resultDTO.setSubType(ObjectUtils.defaultIfNull(approveApplyResDTO.getSubType(), 0));
        resultDTO.setApplyId(applyId);
        return resultDTO;
    }

public static List<DepartmentInfo> sortDepartmentList(List<DepartmentInfo> departmentList) {
        List<DepartmentInfo> sortedDepartmentList = new ArrayList<>();
        // Add the department with deptId = 0 to the sorted list
        for(DepartmentInfo department : departmentList) {
            if(department.getDeptId() == 0) {
                sortedDepartmentList.add(department);
                break;
            }
        }
        // Sort the rest of the departments based on their parent department
        int index = 0;
        while(sortedDepartmentList.size() < departmentList.size()) {
            DepartmentInfo parentDepartment = sortedDepartmentList.get(index);
            index++;
            for(DepartmentInfo department : departmentList) {
                if(department.getParentDeptId() == parentDepartment.getDeptId() && !sortedDepartmentList.contains(department)) {
                    sortedDepartmentList.add(department);
                }
            }
        }
        return sortedDepartmentList;
    }
